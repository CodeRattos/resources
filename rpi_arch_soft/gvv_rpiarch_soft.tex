\section{Office}
\subsection{Document Editor}
\begin{enumerate}[1.]
\item \LaTeX is a powerful word processor typically used for writing mathematical documents. {\em Texmaker} is a very good editor for \LaTeX. {\em Okular} is a nice document viewer for .pdf and other formats and {\em gv} is used for viewing .ps files.
\begin{lstlisting}
sudo pacman -S texlive-most texmaker okular gv
\end{lstlisting}
\item {\em Gnumeric}: Lightweight spreadsheet software.
\begin{lstlisting}
sudo pacman -S gnumeric
\end{lstlisting}
\item {\em Libreoffice}:  MS-Office substitute for Linux.
\begin{lstlisting}
sudo pacman -S libreoffice-fresh
\end{lstlisting}
\end{enumerate}
\subsection{Graphics Editor}
\begin{enumerate}[1.]
\item {\em GIMP}
\begin{lstlisting}
sudo pacman -S gimp  poppler-glib
\end{lstlisting}
\item {\em Inkscape}
\begin{lstlisting}
sudo pacman -S inkscape
\end{lstlisting}
\end{enumerate}
\section{GPIO Software}
The Raspberry Pi has 40 GPIO digital pins for controlling electronic circuits. Related software installation procedure is available below.
\begin{enumerate}[1.]
\item {\em WiringPi}:  Wiring Pi is a simple C library for GPIO programming using the syntax for the Arduino.
\begin{lstlisting}
sudo pacman -S wiringpi
\end{lstlisting}
\item {\em RPI.GPIO}:  Python based GPIO library
\begin{lstlisting}
yaourt -S python-raspberry-gpio
\end{lstlisting}
\item {\em I2C} is a wired communication protocol typically used for monitoring sensor data.
\begin{lstlisting}
sudo pacman -S i2c-tools lm_sensors
sudo nano /boot/config.txt
add
dtparam=i2c_arm=on
at the end of the file. save and exit
\end{lstlisting}
\end{enumerate}
\section{Arduino Software}
\begin{enumerate}[1.]
\item The precompiled {\em Arduino IDE} can be downloaded from the official Arduino website. 
\begin{lstlisting}
sudo usermod -a -G uucp yourregusername
./path/arduino-x.x.x/install.sh
\end{lstlisting}
This will create a arduino icon on the desktop.  In case there is any \textbf{libtinfo} error, while
running an arduino program,
\begin{lstlisting}
sudo pacman -S avrdude
/path/arduino-x.x.x/hardware/tools/avr/bin/avrdude
mv avrdude avrdude_bkup
ln -s /usr/bin/avrdude avrdude
\end{lstlisting}
\item The Python based {\em Platformio} application provides a simple command line interface for programming.
\begin{lstlisting}
sudo pacman -S python-setuptools python-click
yaourt -S platformio
\end{lstlisting}
\item The AVR-Assembly and AVR-GCC software are useful for low level programming 
\begin{lstlisting}
sudo pacman -S avrdude avr-gcc avr-libc
yaourt -S avra
\end{lstlisting}
For {\em avra}, during installation, edit packagebuid using nano, in architecture, give space after x86 and type armv7h
save and exit and continue installing
\end{enumerate}
\section{Math Programming}
{\em Python } provides a powerful alternative to MATLAB through various libraries. 
\begin{lstlisting}
sudo pacman -S geany python-numpy python-scipy python-matplotlib python-mpmath python-cvxopt
\end{lstlisting}
{\em Geany} is an extremely lightweight editor with builtin support for various programming langauges.
